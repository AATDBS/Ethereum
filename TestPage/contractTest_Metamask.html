AYESA Alastria Node - Contract Test Page

Esta página realiza un ejemplo de invocación de un contrato previamente desplegado en la red de Alastria.
Para ello utiliza el provier web3 injectado por el plugin de MetaMask.

Requisitos previos:

  • MetaMask
    Debe tener instalado el plugin de MetaMask y conectado al nodo de red de Alastria.
    RPC personalizado a http://99.80.181.35/rpc.
    Dado que en Alastria el Gas Price es 0, no necesita fondos en su monedero de MetaMask.
    Consulte https://medium.com/crowdbotics/building-ethereum-dapps-with-meta-mask-9bd0685dfd57
  • Contract
    Debe haber desplegado previamente el contrato de pruebas en la red de Alastria y conocer su dirección.
    Para desplegarlo, puede utilizar Remix junto con MetaMask o bien Truffle.
    El código en Solidity del contrato esta diponible en este repositorio público de GitHub.
    No modifique el código del contrato, ya que esta página utiliza el ABI correspondiente al fuente original.
    Consulte Doc
